DEMONSTRATION OF BOX MODEL

BOX-MODEL

*The CSS box model is a box that wraps around every HTML element. It consists of margins, border, padding and the actual content.

*It is used to develop the design and structure of a web page. It can be used as a set of tools to personalize the layout of different components.

*Basically, it is a compartment that includes numerous assets, such as edges, border, padding and material.


THE BOX MODEL CONTAINS DIFFERENT PROPERTIES. SOME OF THEM ARE LISTED BELOW:
  • MARGIN FIELD

    • This segment consists of the area between the boundary and the edge of the border. This margin is transparent. This is the outermost layer, wrapping the content, padding and border as whitespace between this box and other elements.

  • BORDER FIELD

    • It is a region between the padding field and the margin. Its dimensions are determined by the width and height of the boundary.

  • PADDING FIELD

    • It includes the element's padding. This area is actually the space around the content area and within the border area. Its dimensions are given by the width of the padding box and the height of the padding box.

  • CONTENT AREA

    • This area consists of content like text, image, or other media content. It is bounded by the content edge and its dimensions are given by content box width and height.
